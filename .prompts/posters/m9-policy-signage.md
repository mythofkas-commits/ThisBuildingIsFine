# M9 Procedural Poster Prompt Pack - Policy Signage

These prompts are AI-intended source prompts for future replacement assets. The current M9 batch uses deterministic local procedural SVG generation only; no external AI image API or paid tool was used.

## Shared Style

Low-poly/stylized corporate-surreal signage for The Building Is Fine. Clean but wrong, dryly bureaucratic, readable, non-dark, no gore, no monsters, no panic horror, no realistic humans. Use restrained colors, strong text contrast, and simple office-form layout.

## Assets

### m9-policy-poster-hallway

Readable policy poster for a reception hallway. Text: PLEASE MAINTAIN UNCERTAINTY. Supporting copy should feel like calm workplace compliance, not threat.

### m9-policy-poster-meeting

Conference-room policy poster. Text: OPTIONAL MEETINGS MAY ATTEND YOU. Should be funny-unsettling, not scary.

### m9-policy-poster-records

Records-room policy poster. Text: FORMS REMEMBER VERSION HISTORY. Should look like an internal records notice.

### m9-meeting-agenda-card

Meeting agenda card texture. Text: AGENDA: ITEM 1 - YOU, ITEM 2 - MINUTES DISAGREE. Must remain readable from nearby first-person play.
