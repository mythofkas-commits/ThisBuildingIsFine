# Abstract Entities Prompt

Generate or describe abstract non-human surreal entities for The Building Is Fine.

Important:
MVP should not use realistic humans, monsters, combat enemies, or chase creatures. Abstract entities should feel like office systems behaving too personally.

Entity language:
- Meeting invitations.
- Rearranging chairs.
- Agenda items.
- Projector light.
- Exit requests.
- Forms that update themselves.
- Signs that revise the player's certainty.
- Silhouettes only if they read as signage, portraits, shadows, or corporate graphics rather than people.

First hazard:
The Meeting.

The Meeting should:
- Self-schedule.
- Rearrange conference-room furniture.
- Spawn or move agenda displays.
- Create spatial pressure.
- Feel bureaucratic and absurd.
- Avoid monster behavior.
- Avoid jump scares.
- Avoid chase horror.

Possible abstract visuals:
- A projector rectangle that insists on attention.
- Chairs forming quorum around the player.
- Agenda pages sliding into paths.
- A conference table that is too long for the room.
- Door labels changing to meeting topics.

Tone:
- Strange.
- Dryly funny.
- Quietly tense.
- Puzzle-like.
- Not terrifying.
