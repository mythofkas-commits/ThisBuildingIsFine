# Environment Assets Prompt

Generate or describe environment assets for The Building Is Fine.

Asset categories:
- Office rooms.
- Hallways.
- Conference rooms.
- Elevator banks.
- Cubicle areas.
- Break rooms.
- Copy rooms.
- Reception desks.
- Storage rooms.
- Impossible architectural transitions.

Style:
- Low-poly or stylized.
- Clean but slightly wrong.
- Beige carpet, fluorescent panels, glass walls, concrete columns, bland partitions.
- Corporate surfaces with subtle impossible details.

Surreal language:
- Doors labeled with departments that should not exist.
- Hallways that imply disagreement with geometry.
- Exit signs that are too polite.
- Conference rooms with excessive agenda displays.
- Forms placed where signage should be.

Technical preferences:
- Simple geometry.
- Clear scale.
- Modular pieces.
- Repeatable materials.
- Source-controlled metadata.
- Replaceable placeholders.

Avoid:
- Realistic humans.
- Gore.
- Monster shapes.
- Dark haunted architecture.
- Excessive grime.
- Horror lighting.
- Decorative detail that makes navigation unclear.
