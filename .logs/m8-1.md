# M8.1 - Wall Sign Readability Corrective Patch

Date: 2026-05-17

## Summary

Fixed wall-sign flicker and mirrored/backwards readability found during manual M8 playtest. This patch is limited to M8.1 visual/readability correction and preserves the complete five-room run.

## Cause

Wall-mounted sign planes were placed nearly flush with wall surfaces, which could cause z-fighting flicker. Several east/west wall signs also faced the wrong side of the plane, so disabling culling made the room-side view use mirrored backface text.

## Fix

- `createWallSign` now offsets signs `0.045` units along the readable front normal.
- `createWallSign` records `wallOffsetApplied` and `facingNormal` metadata for smoke proof.
- Wall signs now use `backFaceCulling = true` so mirrored backside text is not presented as readable.
- East/west label and wayfinding rotations were corrected so readable fronts face into the room.
- Complete Check-Out status sign now uses the central wall-sign helper.
- Debug/smoke proof now checks decorative wall signs for offset, front-face culling, and representative room-side facing normals.
