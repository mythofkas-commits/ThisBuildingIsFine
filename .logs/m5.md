# M5 - Clarity Meter

Date: 2026-05-16

## Summary

Added a source-driven Clarity system that starts at 100, appears in the HUD, changes subtly from existing play events, and resets on restart. The system is non-punitive in M5 and does not add a fail condition.

## Clarity Events

- Filing an Incident Report lowers Clarity slightly.
- Entering the records room lowers Clarity once.
- Entering the elevator room lowers Clarity once.
- Approaching locked extraction lowers Clarity once.
