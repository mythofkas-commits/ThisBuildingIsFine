# M6 - Performance Review Narrator System

Date: 2026-05-16

## Summary

Added a source-driven Performance Review narrator system. It displays calm bureaucratic messages in the HUD, reacts to existing gameplay events, and uses cooldowns so repeated frame-level checks do not spam the player.

## Narrator Events

- Initial performance review pending.
- Incident Report collection.
- Clarity change.
- Locked extraction attempt.
- Extraction approval.
- Extraction completion.
- Restart.
- Selected room entry.

## Rate Limiting

- Each narrator event has its own `cooldownFrames` value.
- A small global frame cooldown prevents multiple non-forced messages from stacking in the same moment.
- Important one-shot milestone events can force emission so smoke proof and player feedback remain deterministic.
