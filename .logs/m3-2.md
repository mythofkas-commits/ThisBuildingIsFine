# M3.2 - Stability And Regression Guardrails

Date: 2026-05-16

## Scope

Stability patch only. Added behavior contracts, static validation, package-level verification, and non-brittle smoke checks for the accepted M1-M3 foundation. Did not add extraction, win behavior, Clarity, Performance Review, The Meeting, AI assets, enemies, combat, inventory, multiplayer, or new gameplay mechanics.

## Stability Protections Added

- `STABILITY.md` defines locked behavior contracts for:
  - movement and camera,
  - restart flow,
  - five-room modular layout,
  - room connectivity,
  - collision and walkable behavior,
  - Incident Report system,
  - HUD behavior,
  - tone constraints,
  - AI-only workflow.
- `scripts/validate-project-state.mjs` validates required source files, required room IDs, required report IDs, npm scripts, stack boundaries, and premature future-milestone feature files.
- `npm run validate` runs the static validator.
- `npm run verify` runs `check`, `build`, `validate`, and `smoke`.
- `AGENTS.md` now instructs future milestones to preserve `STABILITY.md` contracts.
- `TASKS.md` now requires `npm run verify` before future milestones claim pass.

## Smoke Updates

The smoke test now checks behavior contracts without relying on exact mesh counts, exact report coordinates, exact colors, or screenshot pixels:

- scene loads,
- HUD exists,
- at least five rooms are detected,
- required room IDs are present,
- movement works,
- room traversal movement works,
- restart works,
- wall collision behavior exists,
- intended doorway probes remain clear,
- three required Incident Reports exist,
- reports start visible,
- collecting a report updates HUD count,
- restart restores report visibility/count,
- console errors and page errors are absent.

## Protected Against

- Accidental deletion of core source files.
- Accidental removal of required rooms or Incident Reports.
- Replacing the source-driven Babylon/Vite workflow with Unity, Godot, or Unreal project files.
- Premature Clarity, Performance Review, The Meeting, extraction/win, inventory, combat, enemy, or multiplayer feature files before their approved milestone.
- Regressions where scene load, HUD, movement, traversal, restart, report collection, or report reset silently break.
